Clarkson is a modern set of tools to take WordPress theme development without losing touch with WordPress.

Clarkson Theme

  • Twig
  • Gulp
  • BrowserSync
  • NPM

Clarkson Core

  • Twig
  • WordPress objects

At Level Level we created an Open Source plugin ‘Clarkson Core’: a Twig based template engine with under the hood a collection of Object-oriented WordPress classes and some lightweight handy features. This, in combination with a basic modern starter-theme and some agreed upon guidelines, contributes to a workflow that keeps comprehensive projects aligned with your high technical goals. Meanwhile keeping the WordPress way of development in mind.
It's the missing WordPress theming layer.
